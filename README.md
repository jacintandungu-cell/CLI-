# Project Tracker CLI

A command-line application to manage **users, projects, and tasks**.  
This tool helps organize work by allowing you to add users, assign projects, create tasks with deadlines, and mark them complete.  
Data is stored in a simple JSON file for persistence — no database setup required.

---

## ✨ Features
- **User Management**: Add and list users with names and emails.
- **Project Management**: Assign projects to users and list them.
- **Task Management**: Add tasks with due dates, list tasks per project, and mark tasks as complete.
- **Persistent Storage**: All data is saved in `project_data.json`.
- **Colorful Output**: Uses [Rich](https://github.com/Textualize/rich) for styled, readable CLI output.

---

## 📦 Requirements
- Python 3.12+
- pip (installed with `get-pip.py`)
- Dependencies:
  - `rich`
  - `python-dateutil`

Install dependencies:
```bash
pip install --user rich python-dateutil --break-system-packages
